🌐 Cara Onlinekan NexMart (Gratis & Cepat)

Karena jalur tunnel mengalami kendala, kita akan menggunakan Netlify Drop. Ini adalah cara termudah di dunia untuk mengonlinekan website.

Langkah 1: Buka Netlify Drop

Klik tombol di bawah ini untuk membuka halaman upload Netlify:

Buka Netlify Drop ↗

Langkah 2: Siapkan Folder Proyek

Buka File Explorer Anda (Windows + E), lalu pergi ke:
Local Disk (C:) > marketplace_v2

Langkah 3: Tarik dan Lepas (Drag & Drop)

Klik dan tahan folder marketplace_v2, lalu tarik dan lepaskan ke kotak yang ada di website Netlify tadi.

TARIK FOLDER KE SINI

Langkah 4: Selesai!

Tunggu beberapa detik hingga proses upload selesai. Netlify akan memberikan Anda link website (misal: shiny-unicorn-123.netlify.app).

💡 Tips Selanjutnya:

Setelah Anda mendapatkan link-nya, kirimkan link tersebut kepada saya. Saya akan terus melanjutkan koding di sini, dan Anda tinggal mengulang proses Drag & Drop ini jika ingin memperbarui tampilan online-nya.

Cara ini jauh lebih stabil dan bisa diakses oleh siapa saja di seluruh dunia!
